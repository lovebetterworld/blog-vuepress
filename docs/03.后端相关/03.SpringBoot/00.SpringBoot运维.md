## 1 运行脚本

`vi startup.sh`

```bash
#!/bin/bash
PID=$(ps -ef | grep springboot.jar | grep -v grep | awk '{ print $2 }')
if [ -z "$PID" ]
then
    echo '服务已经启动！！！'
else
    echo '停止服务！'
    kill $PID
    nohup java -jar springboot.jar --spring.profiles.active=prod > /root/logs/springboot.log &
    echo '查看日志'
    tail -f /root/logs/springboot.log
fi
```

脚本授权：

```bash
chmod +x startup.sh
```
